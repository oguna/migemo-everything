# migemo-everything

![](screenshot.png)

## ビルド方法

```shell
cargo build --release
```

## 実行方法
実行ファイルと同じフォルダに、`Everything64.dll`と`migemo-compact-dict`があることを確認してください。
また、Everythingが実行中であることも確認してください。

`migemo-everything.exe`を起動すると、画面が表示されます。
